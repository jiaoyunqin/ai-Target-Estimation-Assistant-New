# 一期会话与导出能力说明

## 1. 本期结论

一期不做版本管理。

删除独立【版本与协同】模块，也不在右侧边栏展示个人版本、正式版本、保存版本、发布版本等能力。

本期只保留：

- 资管用户发起会话
- 新建一个大促
- 按模块填写信息并发起测算
- 大盘测算导出 CSV / 报告
- 行业测算导出 CSV / 报告

## 2. 模块范围

### 2.1 大促信息

用户在该模块维护大促基础信息，包括：

- 大促基本信息
- 阶段拆分
- Big Day
- 竞对信息
- 预算规划
- 历史参考与可比日

本期大促信息不支持导出。

### 2.2 大盘测算

用户填写 OKR 或必要目标信息后，直接发起大盘测算。

大盘测算完成后，右侧边栏提供：

- 导出 CSV
- 导出报告

不提供版本保存、版本发布、版本引用。

### 2.3 行业测算

行业测算支持两种输入方式：

- 继承本人之前已经测算好的大盘测算内容
- 手动输入必要信息

选择任一方式后，可以发起行业测算。

行业测算完成后，右侧边栏提供：

- 导出 CSV
- 导出报告

不提供版本保存、版本发布、版本引用。

## 3. 右侧边栏规则

右侧边栏只保留当前模块必要能力。

大促信息：

- 下游引用说明
- 不展示导出
- 不展示版本

大盘测算：

- 自动获取 Skill
- 测算步骤状态
- 大盘测算导出

行业测算：

- 自动获取 Skill
- 测算步骤状态
- 行业测算导出

## 4. 二期再做

以下能力放到二期：

- 个人版本
- 正式版本
- 版本列表
- 版本引用
- 跨用户版本共享
- 统一版本与协同中心

## 5. 验收标准

- 顶部导航不展示【版本与协同】。
- 三大模块右侧边栏不展示版本管理。
- 大促信息不展示导出入口。
- 大盘测算展示导出 CSV / 导出报告。
- 行业测算展示导出 CSV / 导出报告。
- 行业测算可选择继承本人历史大盘测算或手动输入。
